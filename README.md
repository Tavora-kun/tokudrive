O **TokuDrive Max** nasceu de uma necessidade real da comunidade de fãs: um ecossistema completo, rápido e sem anúncios para assistir a Tokusatsu e Anime em altíssima qualidade. Esqueça os reprodutores de navegador que travam ou não suportam formatos pesados. Nós trouxemos a qualidade de cinema para o seu desktop.

## ✨ Funcionalidades Principais

* 🚀 **Motor de Vídeo MPV Nativo:** Reproduza ficheiros MKV pesados, HEVC/H.265 e formatos de áudio complexos de Fansubs sem engasgos. Qualidade de cinema em ecrã inteiro.
* 📡 **Radar Multisub P2P (Nyaa):** Um sistema de busca inteligente integrado diretamente aos maiores trackers da comunidade. Encontre os melhores lançamentos, projetos com múltiplas legendas (Multisubs) e baixe/assista via streaming P2P instantâneo.
* 🗄️ **Integração com Servidores Privados:** Conecte o seu servidor **Jellyfin** ou **Emby** e aceda a todo o seu acervo pessoal com um visual premium, posters em HD e carregamento direto.
* 📺 **Gestor de Temporadas (Batches):** Encontrou um torrent com uma série inteira? O TokuDrive Max lista todos os episódios de forma organizada para você escolher exatamente qual quer assistir.
* 🎨 **Interface Premium Glassmorphism:** Um design moderno, escuro e imersivo, feito para não cansar a vista durante as suas maratonas.

## 💻 Instalação (Utilizadores Finais)

A forma mais fácil de usar o TokuDrive Max é baixar a versão compilada na aba [Releases](../../releases) do GitHub.

### Para Windows
1. Vá até à página de **Releases**.
2. Baixe o ficheiro `TokuDrive Max Setup 1.0.0.exe`.
3. Dê um duplo clique para instalar (aceite os Termos de Uso).
4. Abra o aplicativo a partir do seu Menu Iniciar ou Ambiente de Trabalho e divirta-se!

### Para Linux
1. Vá até à página de **Releases** e baixe o ficheiro `.AppImage`.
2. Certifique-se de que tem o motor MPV instalado no seu sistema base (ex: `sudo apt install mpv` no Ubuntu/Mint).
3. Clique com o botão direito no `.AppImage` > Propriedades > Permissões > Marque "Permitir execução do ficheiro como um programa".
4. Dê um duplo clique para abrir!
